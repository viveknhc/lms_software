# Software Requirements Specification (SRS)

## Introduction

A Learning Management System supporting online education.

## Functional Requirements

-   Authentication
-   Course Management
-   Enrollment
-   Learning Content
-   Assessments
-   Assignments
-   Certificates
-   Payments
-   Analytics
-   Notifications

## Non-Functional Requirements

-   Security
-   Scalability
-   Performance
-   Availability
-   Maintainability

## User Roles

-   Student
-   Instructor
-   Administrator

## Technology Stack

-   Django
-   Django REST Framework
-   PostgreSQL
-   Redis
-   Docker
